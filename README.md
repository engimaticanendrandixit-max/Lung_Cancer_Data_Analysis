# 🫁 Lung Cancer Data Analysis

## 📌 Overview

This project analyzes a lung cancer dataset to explore patterns related to cancer stages, treatment selection, smoking habits, age at diagnosis, and multimorbidity among patients.

The analysis is performed using Python and includes graphical visualizations and statistical summaries.

---

## 🎯 Project Objectives

* Study the relationship between age and cancer stage.
* Compare cancer stage distributions across countries.
* Analyze treatment selection patterns.
* Examine the impact of smoking on age at diagnosis.
* Measure multimorbidity prevalence.

---

## 🛠️ Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

---

## 📊 Tasks Performed

### Task 1: Age at Diagnosis vs Cancer Stage

* Calculated mean and median age for each cancer stage.
* Compared age distributions.
* Generated boxplots and violin plots.

---

### Task 2: Country-wise Cancer Stage Distribution

* Computed percentage distribution of cancer stages across countries.
* Visualized country-wise differences using bar charts.

---

### Task 3: Treatment Type Selection by Cancer Stage

* Studied treatment frequencies across cancer stages.
* Compared surgery, chemotherapy, radiation, and combined treatments.

---

### Task 4: Smoking Status vs Age at Diagnosis

* Compared age distributions across smoking categories.
* Generated summary statistics and boxplots.

---

### Task 5: Multimorbidity Prevalence Analysis

* Combined hypertension, asthma, cirrhosis, and other cancer.
* Computed illness counts and analyzed multimorbidity.
